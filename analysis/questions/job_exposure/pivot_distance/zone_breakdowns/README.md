# Zone Breakdowns

Per-job-zone detail on which skills and knowledge elements drive the pivot cost (absolute imp×level units). Four figures per zone:

- `top_skills_gained.png` — top 10 skills the worker must acquire
- `top_knowledge_gained.png` — top 10 knowledge areas the worker must acquire
- `top_skills_dropped.png` — top 5 skills the worker would leave behind
- `top_knowledge_dropped.png` — top 5 knowledge areas the worker would leave behind

Computed from `pivot_distance/results/element_costs_by_zone.csv`. Generated by `pivot_distance/run.py`.
