## Methodology Notes

All results in this report are generated by the same compute pipeline that powers the AEA Dashboard web application. This guarantees consistency between what appears in the interactive tool and what is reported here.

**Core metrics:**
- **% Tasks Affected** — share of an occupation's weighted task completion attributable to AI-exposed tasks (ratio-of-totals, not average-of-percentages)
- **Workers Affected** — `(% Tasks Affected / 100) x total employment`
- **Wages Affected** — `(% Tasks Affected / 100) x employment x median annual wage`

**Data sources:** Anthropic Economic Index (AEI), MCP Server Pipeline, Microsoft Copilot Analysis, O*NET (2025/2015), BLS OEWS 2024.

**Default configuration** unless noted otherwise: all snapshot datasets (AEI v1-v4, AEI API v3-v4, MCP v4, Microsoft), Average combination, frequency-weighted method, all tasks included, auto-aug multiplier enabled, national geography.

See [ARCHITECTURE.md](../ARCHITECTURE.md) for full computation details and [PRD.md](../PRD.md) for metric definitions.
